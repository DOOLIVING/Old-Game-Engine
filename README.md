### 🌐 Компиляция под Web
Движок теперь полностью поддерживает запуск в браузере.
* **Web-ядро**: Переработана архитектура ядра для бесшовной работы в веб-среде.
* **Запуск**: Для запуска веб-версии в VS Code нажмите **Go Live**.
* **Оптимизация**: Адаптирована загрузка ресурсов и обработка ввода для веб-контекста.

### 🛠️ Web-редактор (Beta)
Для веб-версии разработан встроенный редактор сцены.
* **Текущее состояние**: Функционал реализован на **60%** — доступны базовые манипуляции с объектами, настройка параметров материалов и предпросмотр изменений в реальном времени.
* **В разработке**: Сохранение/загрузка сцен, импорт пользовательских моделей и расширенная работа с шейдерами через веб-интерфейс.
* **Доступность**: Редактор активируется через специальный режим при запуске веб-сборки.

### 🎨 Обновленная графика
Графический конвейер был значительно улучшен.
* Повышено качество рендеринга и освещения.
* Оптимизирована работа шейдеров для веб-платформы.

### ⚡ Реалистичная физика
Модуль физики претерпел серьёзные изменения.
* **Улучшенная симуляция**: Поведение объектов стало более реалистичным и точным.
* Переработаны алгоритмы коллизий и расчёта импульсов.

### 🖥️ Прототип интерфейса
Добавлен прототип пользовательского интерфейса.

### ⚡ Шейдерная архитектура (GLSL)
Движок теперь поддерживает кастомные конвейеры отрисовки через `PShader`.
* **Класс `ShaderManager`**: Централизованное управление `.glsl` файлами.
* **Эффекты**: Реализованы базовые шейдеры постобработки (Bloom, CRT-эффект для ретро-стилистики) и шейдеры материалов (Normal Mapping).
* **Динамика**: Возможность передавать переменные (uniforms) в шейдер в реальном времени (например, время или позицию игрока).

### 🍎 Физический движок (Internal Physics)
Добавлен модуль обработки физики, обеспечивающий реалистичное поведение объектов.
* **Система коллизий**: Реализована проверка столкновений типа **AABB** (Axis-Aligned Bounding Box) для кубов и сфер.
* **Гравитация и импульсы**: Объекты теперь имеют массу и могут взаимодействовать с «землей» и друг другом.
* **Raycasting**: Базовая поддержка пускания лучей для определения дистанции или выбора объектов в сцене.

### 🚗 Работа с моделями и текстурами
* **`.obj` Import**: Полноценная загрузка геометрии с корректным масштабированием.
* **UV-Mapping**: Улучшенная работа с текстурами на сложных мешах.
