# 🕹️ Old Game Engine

**Old Game Engine** — это минималистичный 3D-движок, написанный с нуля на Java - Processing.  
Он создан для быстрого прототипирования игр и 3D-сцен с поддержкой освещения, звука, UI и системы сцен.

> Движок не навязывает готовое управление — вы сами решаете, как использовать камеру, физику и логику.

---

## ✨ Особенности

- 🧊 **Базовые 3D-примитивы**: куб, сфера, эллипс, прямоугольник
- 🎨 **Поддержка 3D-моделей**: загрузка и отображение `.obj` файлов
- 🖼️ **Текстурирование**: наложение изображений на примитивы
- 💡 **Динамическое освещение**: до 8 источников света с визуализацией
- 🎥 **Свободная камера**: позиция + углы `yaw`/`pitch` (под вашу логику)
- 🖱️ **UI-кнопки**: с обработкой кликов поверх 3D
- 🔊 **Звуковая система**: фоновая музыка и эффекты (любые форматы)
- 🎬 **Система сцен**: легко переключаться между меню и уровнями
- ⚙️ **Флаг физики** у объектов (готов к расширению)
- ✨ **Система частиц**: для визуальных эффектов

## 🆕 Что нового (обновление от 8 апреля 2026)

### 🚗 Поддержка 3D-моделей
- Добавлен класс `model` для загрузки и отображения `.obj` файлов

- Класс `cube` теперь поддерживает наложение текстур
- Параметр `boolean useTexture` для переключения между цветом и текстурой
- Автоматическая загрузка изображений через `loadImage()`

- Новый класс `ParticleSystem` и `Particle`
- Создание визуальных эффектов (дым, искры, магические эффекты)
- Настраиваемые параметры: время жизни, скорость, гравитация

### 🎮 Улучшенное управление
- Поддержка как латинской, так и русской раскладки клавиатуры
- Раздельное управление камерой и объектами на сцене
- Пример управления автомобилем в реальном времени

## 🚀 Установка и запуск

1. Установите [Processing](https://processing.org/download)
2. Добавьте библиотеку **Sound**:
   - `Sketch → Import Library → Add Library → Sound`
3. Скопируйте все файлы движка в одну папку
4. Поместите свои ресурсы в папку `data/`:
   - Аудиофайлы (`.mp3`, `.wav`)
   - 3D-модели (`.obj`)
   - Текстуры (`.jpeg`, `.png`)
5. Откройте основной `.pde` файл и нажмите **Run**

## 🎮 Пример: ваша первая сцена

### Базовая сцена с текстурированным кубом и частицами
```java
class MyScene extends Scene {
  MyScene() {
    bgColor = 50;
    
    // Загружаем текстуру
    PImage texture = loadImage("texture.jpg");
    
    // Текстурированный куб
    worldCubes.add(new cube(0, 0, 0, 50, 255, 0, 0, true, texture));
    
    // Система частиц в центре
    worldParticles.add(new ParticleSystem(new PVector(0, 0, 0)));
    
    // Источник света
    worldLight.add(new light(255, 255, 255, 100, 100, 100));
    
    // Фоновая музыка
    worldSound.add(new sound("music.mp3", 0.5));
    worldSound.get(0).loop();
  }

  @Override
  void update() {
    super.update();
    // ваша логика здесь
  }
}
```

### Сцена с 3D-моделью и управлением
```java
class CarScene extends Scene {
  model myCar;
  float carX = 0;
  
  CarScene() {
    bgColor = 100;
    
    // Загружаем 3D-модель
    myCar = new model(carX, 0, 0, 50, "car.obj");
    myCar.rotX = radians(180); // Корректировка ориентации
    worldModels.add(myCar);
    
    // Добавляем эффект частиц за машиной
    worldParticles.add(new ParticleSystem(new PVector(carX, 0, -20)));
  }
  
  @Override
  void update() {
    super.update();
    movement(); // Ваша функция управления
  }
  
  void movement() {
    if (keyPressed) {
      if (key == 'r' || key == 'к') {
        carX += 2.0;
        myCar.x = carX; // Обновляем позицию модели
      }
    }
  }
}
```

А в главном скетче переключиться на неё:

```java
currentScene = new MyScene();
```

## 🧰 Классы движка

| Класс | Описание |
|-------|----------|
| `Camera` | Позиция + углы, метод `apply()` |
| `Scene` | Базовый класс сцены (списки объектов, отрисовка) |
| `button` | UI-кнопка с текстом и состоянием |
| `light` | Источник света (RGB + позиция) |
| `cube` / `sphere` / `ellipse` / `rect` | 3D-примитивы с поддержкой текстур |
| `model` | Загрузка и отображение `.obj` файлов |
| `ParticleSystem` / `Particle` | Система частиц для эффектов |
| `sound` | Обёртка над SoundFile (play, loop, stop, pause) |

## 🎯 Управление в примере GameLevel

| Действие | Клавиши (EN) | Клавиши (RU) |
|----------|-------------|--------------|
| Камера влево/вправо | `A` / `D` | `Ф` / `В` |
| Камера вперед/назад | `W` / `S` | `Ц` / `Ы` |
| Камера вверх/вниз | `Q` / `E` | `Й` / `У` |
| Поворот камеры | `←` `→` `↑` `↓` | `←` `→` `↑` `↓` |
| Движение машины | `R` | `К` |
