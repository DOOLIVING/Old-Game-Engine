# 🕹️ VertexEngine

[![Version](https://img.shields.io/badge/version-v2.1.1--stable-green.svg)](https://github.com/your-repo)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**VertexEngine** — это легковесный 3D-движок на Java (Processing), вдохновленный эстетикой начала 2000-х. Теперь это не просто графическая оболочка, а функциональная среда с поддержкой шейдеров и физического взаимодействия.

---

## 🚀 Новое в версии 2.1.1 (Апрель 2026)

### 🖥️ Прототип интерфейса
Добавлен прототип пользовательского интерфейса.

### ⚡ Шейдерная архитектура (GLSL)
Движок теперь поддерживает кастомные конвейеры отрисовки через `PShader`.
* **Класс `ShaderManager`**: Централизованное управление `.glsl` файлами.
* **Эффекты**: Реализованы базовые шейдеры постобработки (Bloom, CRT-эффект для ретро-стилистики) и шейдеры материалов (Normal Mapping).
* **Динамика**: Возможность передавать переменные (uniforms) в шейдер в реальном времени (например, время или позицию игрока).

### 🍎 Физический движок (Internal Physics)
Добавлен модуль обработки физики, обеспечивающий реалистичное поведение объектов.
* **Система коллизий**: Реализована проверка столкновений типа **AABB** (Axis-Aligned Bounding Box) для кубов и сфер.
* **Гравитация и импульсы**: Объекты теперь имеют массу и могут взаимодействовать с «землей» и друг другом.
* **Raycasting**: Базовая поддержка пускания лучей для определения дистанции или выбора объектов в сцене.

### 🚗 Работа с моделями и текстурами
* **`.obj` Import**: Полноценная загрузка геометрии с корректным масштабированием.
* **UV-Mapping**: Улучшенная работа с текстурами на сложных мешах.

## ✨ Основные возможности

* **💡 Освещение**: До 8 динамических источников света с расчетом затухания.
* **🎥 Камера**: Свободный полет (Yaw/Pitch) с поддержкой WASD и стрелок.
* **✨ Частицы**: Система VFX для создания эффектов огня, дыма и искр.
* **🔊 Звук**: Многоканальная аудио-система (музыка + SFX).
* **⌨️ Ввод**: Полная поддержка RU/EN раскладок без конфликтов.

## 🛠️ Структура проекта

```text
📂 OldGameEngine/
├── 📄 main_sketch.pde  <-- Точка входа
├── 📄 physics.pde      <-- [NEW] Коллизии, гравитация, лучи
├── 📄 shaders.pde      <-- [NEW] Загрузка и управление GLSL
├── 📄 model.pde        <-- Загрузчик .obj моделей
├── 📄 particles.pde    <-- Генератор частиц (VFX)
├── 📄 primitives.pde   <-- Геометрия и текстурирование
└── 📂 data/
    ├── 📂 shaders/     <-- .glsl файлы (vertex/fragment)
    ├── 📂 models/      <-- .obj файлы
    └── 📂 textures/    <-- Текстуры объектов
```

## 🏎️ Быстрый старт

### Включение физики для объекта
```java
cube myBox = new cube(0, 100, 0, 50);
myBox.hasPhysics = true; // Теперь куб будет падать и сталкиваться
worldCubes.add(myBox);
```

### Применение шейдера
```java
PShader pixelated = loadShader("pixel.glsl");
shader(pixelated); // Применить ко всей сцене
```
